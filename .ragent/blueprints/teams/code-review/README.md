# Code Review Team Example

A 3-teammate review swarm:

- security reviewer
- performance reviewer
- test reviewer

Use these prompts with `team_spawn` after creating `code-review`.

## Quick run

1. Create the team:

```text
/team create code-review
```

2. Spawn teammates using prompts in `spawn-prompts.md`.

3. Add review tasks using `task-seed.json`.

4. Monitor progress:

```text
/team status
/team tasks
```
