# ragent: AI Coding Agent - Introduction and Overview

ragent is an AI coding agent designed for terminal-based development, built entirely in Rust and inspired by OpenCode. It provides developers with a powerful tool for automating coding tasks through natural language interactions, combining multi-provider LLM orchestration, a comprehensive tool system, and a terminal-based user interface. The system operates as an intelligent assistant that understands codebase context and performs complex tasks ranging from code explanation and refactoring to bug detection and feature implementation.

The platform offers extensive features including native integration with major LLM providers (Anthropic, OpenAI, GitHub Copilot, Ollama), a comprehensive tool system with 26 built-in tools spanning file operations, web integration, task management, and document processing, and a full-screen terminal UI built with ratatui. Additional capabilities include HTTP server architecture with REST + SSE API, customizable agent presets, MCP client support, file snapshotting, and background agent spawning for parallel execution.

ragent targets professional developers, technical leads, DevOps engineers, students, educators, and open source contributors. Key benefits include enhanced productivity through automation of routine tasks, improved code quality via built-in review capabilities, accelerated learning for newcomers to codebases, reduced cognitive load by handling mechanical coding aspects, and flexible integration through both terminal UI and HTTP API. The documentation is organized into seven sections covering getting started, core concepts, advanced features, API reference, guides, troubleshooting, and architecture.
