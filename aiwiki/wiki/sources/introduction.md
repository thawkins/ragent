# ragent Introduction and Overview

ragent is an AI coding agent designed for the terminal, built entirely in Rust and inspired by OpenCode. It provides developers with a powerful tool for automating coding tasks through natural language interactions, combining multi-provider LLM orchestration, a comprehensive tool system, and a terminal-based user interface. The tool operates as an intelligent assistant that understands codebase context and can perform complex tasks ranging from code explanation and refactoring to bug detection and feature implementation.

ragent offers extensive features including multi-provider LLM support (Anthropic, OpenAI, GitHub Copilot, Ollama), a comprehensive tool system with 26 built-in tools covering file operations, web integration, task management, and sub-agent capabilities. It features a full-screen ratatui terminal interface, HTTP server architecture with REST + SSE API, customizable agent presets, Model Context Protocol client support, file snapshotting, and background agent spawning. The documentation is structured to support both quick reference and comprehensive learning, covering installation, core concepts, advanced features, API reference, guides, troubleshooting, and architecture.
