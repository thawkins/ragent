# AIWiki User Guide

AIWiki is an embedded, project-scoped knowledge base system designed for the ragent environment. It automatically compiles source documents into an interconnected wiki of markdown pages with features including automatic cross-linking, incremental updates, and a web-based interface. The system organizes knowledge into four page types: entities (people, organizations, places), concepts (ideas, topics, theories), sources (document summaries), and analyses (derived content like comparisons).

The guide covers installation via slash commands, document ingestion from multiple formats (Markdown, PDF, Word, OpenDocument), and various tools for agents to search, ingest, and export wiki content. AIWiki supports bidirectional linking between related pages, tracks file changes for efficient syncing, and provides both a local web interface at localhost:9100/aiwiki and export capabilities to formats like single markdown and Obsidian vaults. Best practices include organizing files in subdirectories, using kebab-case naming, and following a sync workflow for maintaining the knowledge base.
