# AIWiki User Guide

AIWiki is an embedded, project-scoped knowledge base system designed for the ragent environment. It automatically compiles source documents into an interconnected wiki of markdown pages featuring automatic cross-linking and incremental updates. The system processes various document formats including Markdown, plain text, PDF, Word, and OpenDocument files, extracting entities and concepts to generate four types of wiki pages: Entities (people, organizations, places), Concepts (ideas, topics, theories), Sources (document summaries), and Analyses (derived content like comparisons).

The system provides multiple interfaces for interaction including slash commands for initialization, syncing, and management; agent tools for programmatic access; and a web interface for browsing, searching, editing, and visualizing page relationships. AIWiki supports import/export functionality including Obsidian-compatible vaults and single-file markdown exports. Key features include incremental processing based on file hashes, bidirectional cross-linking between related pages, and the ability to disable when not needed for zero performance impact.
