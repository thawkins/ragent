# ragent TUI Keyboard Shortcuts Reference

This document provides a comprehensive reference for keyboard shortcuts in the ragent TUI (Terminal User Interface) application. It covers global shortcuts that work across all contexts, specialized shortcuts for different views including message view, output view, and file menu, as well as dialog-specific shortcuts for memory browser, plan approval, provider setup, LSP edits, and context menus. The document establishes navigation standards using arrow keys as the primary method, with Esc for dismissal and Enter for confirmation as consistent patterns. Implementation notes indicate the codebase location in crates/ragent-tui/ and guidelines for adding new shortcuts.
