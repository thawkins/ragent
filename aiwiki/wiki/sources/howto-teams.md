# Teams How-To Manual for ragent

This comprehensive guide provides practical, end-to-end instructions for using the Teams capability in ragent, a multi-agent collaboration system. Teams allow users to orchestrate multiple AI teammates working together on complex tasks through both TUI commands and tool-driven workflows. The document covers team architecture with project-local and global storage options, quick start procedures using blueprints (pre-built team templates), detailed TUI command workflows, and fully programmatic tool-based workflows. Key concepts include team creation with contextual prompts, teammate spawning and lifecycle management, shared task lists with dependency tracking, real-time monitoring through the Teams panel, and safe cleanup operations. The manual emphasizes best practices such as always providing context when creating teams, using blueprints for repeatable patterns, proper sequencing of wait operations, and conservative lifecycle management to avoid disrupting active teammate sessions.
