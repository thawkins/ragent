# LSP Integration Strategies

### From: LSP_INTEGRATION_GUIDE

Three approaches for adding LSP capabilities: Native Tool (built-in tool in tool/lsp.rs), MCP Server (wrapping LSP client as MCP configuration), or Hybrid (combining both for different features). Each leverages existing tool registration and event systems.
