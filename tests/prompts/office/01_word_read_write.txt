# Word Document Read/Write Test
# Tests the office_read and office_write tools for .docx files

## Test 1: Create a new Word document

Use office_write to create a new Word document at target/temp/test_document.docx with the following content:
- Title: "Test Report"
- A paragraph with introductory text
- A heading "Section 1: Overview"
- Another paragraph with sample content
- A heading "Section 2: Data"
- A bulleted list with 3 items

## Test 2: Read the created document

Use office_read to read the document created in Test 1:
1. Read with default format (markdown)
2. Verify all sections are present
3. Check that headings are properly formatted

## Test 3: Get document metadata

Use office_info to retrieve metadata about the document:
1. Check the document type
2. Verify any available metadata (author, creation date, etc.)

Expected behavior:
- office_write should create a valid .docx file
- office_read should return the content in readable markdown format
- office_info should return structural information about the document
