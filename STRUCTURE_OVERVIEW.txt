     → Feed back to LLM

     → Agent prompt + working_dir

     → AGENTS.md contents
     → Available skills
     → Tool guidelines
     → File reading practices

     MessagePart::Text { text }
     MessagePart::ToolCall { tool, call_id, state: ToolCallState }
     MessagePart::Reasoning { text }
